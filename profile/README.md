# RapidFort automatically hardens your containers and removes vulnerabilities


[![RF Hardened][rf-h-badge]][rf-link-hardened-badge]
[![Dockerhub][dh-rf-badge]][dh-rf]
[![Slack][slack-badge]][slack-link]
[![License][license-badge]][license]


RapidFort drastically reduces OSS vulnerability and patch management burden so that developers can focus on building. By eliminating unused container components, RapidFort enhances production workload security and saves developers from unnecessarily investigating, patching and maintaining unused code.

To read more about the exciting work we're doing, check out the [RapidFort Blog][rf-blog-link]. If you're interested in working with us, see the [RapidFort Careers][rf-careers-link] page.

[![RapidFort][rf-vulns-ui-image]][rf-vulns-ui-link]

# How to build hardened containers

We use RapidFort's Software Attack Surface Management platform to optimize and harden the community images. You can contribute to the project by using the hardened images and providing feedback, improving the coverage scripts and regression tests for existing images, requesting the addition of new images, or by rolling up your sleeves and hardening new images using RapidFort's free tier product. Let's make the OSS world safer together.

[![Build Hardened Containers][rf-build-hardened-image]][rf-build-hardened-link]

# Community Images

RapidFort is excited to launch our open-source Community Images project, securing and hardening the world's most popular container images. Join us, use the hardened images, comment, contribute, and help reduce the software attack surface of billions of open-source container instances.
